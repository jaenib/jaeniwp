Overview

Dream Together

Open-source image generation as a public archive of voices.

Dream Together is a contribution to the Tbilisi Architecture Biennial 2022. The project uses open-source image generation to visualise narratives from diverse communities and to publish them as a searchable archive.

Approach

From prompts to public archive

  • Collaborative prompt building with participants and organizers.
  • Open-source image generation to keep the pipeline transparent.
  • Web archive to surface voices and allow visitors to browse themes.

Outputs

Image archive, installation visuals, public web portal
